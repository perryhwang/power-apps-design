# Power Apps权限审批系统 - UI设计效果图

## 整体设计理念

**设计风格**：现代简约、企业级专业风格  
**主色调**：深蓝色(#2B579A) + 白色 + 浅灰色  
**辅助色**：绿色(成功)、橙色(警告)、红色(错误)  
**字体**：Microsoft YaHei、Segoe UI

## 设计规范

**颜色规范**：
- 主色：#2B579A (深蓝)
- 成功：#28A745 (绿色)
- 警告：#FFC107 (橙色)  
- 错误：#DC3545 (红色)
- 背景：#F8F9FA (浅灰)

**图标使用**：
- 📊 仪表板/统计
- 📤 导入/上传
- 📋 申请/列表
- 👥 用户/审核人
- 🔍 搜索/筛选
- ⚙️ 设置/配置
- ✅ 成功/通过
- ❌ 失败/拒绝
- ⏳ 进行中
- ⚠️ 警告/超时

**响应式布局**：
- 桌面端：1200px+ (完整布局)
- 平板端：768-1199px (紧凑布局)
- 手机端：<768px (堆叠布局)
